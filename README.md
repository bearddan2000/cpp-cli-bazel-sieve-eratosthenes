# cpp-cli-bazel-sieve-eratosthenes

## Description
Given a number find all primes.

## Tech stack
- cpp
- bazel

## Docker stack
- l.gcr.io/google/bazel:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
